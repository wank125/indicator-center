#### KPI卡片组件 (`IndicatorKpiCard.vue`)

```vue
<template>
  <div class="kpi-card">
```

**Props**: `label: string`, `code: string`, `unit: string`, `compareType: 'yoy'|'mom'`

**数据加载**: 调用 `GET /api/indicator/data/daily?indicator_code={code}&latest=1` 获取当日值，同时取同比/环比数据。

**数值格式化规则**:
- 电量: `1,234.5` (千分位+1位小数)
- 电价: `312.50` (2位小数)
- 电费: `256,780` (千分位取整)
- 百分比: `78.5%`
- 小时: `4,521.3`

#### 组件拆分

| 组件 | 文件 | 说明 |
|------|------|------|
| `IndicatorTree.vue` | 左侧树形菜单 | 支持搜索、3种维度切换、懒加载 |
| `IndicatorToolbar.vue` | 查询工具栏 | 指标选择器+时间范围+粒度切换 |
| `IndicatorChart.vue` | 图表区域 | 根据指标类型自动选择图表类型 |

| `IndicatorMultiQuery.vue` | 多指标叠加 | 支持最多5个指标在同一图表对比 |

#### 关键交互

1. **点击左侧树节点** → 自动填充指标编码到工具栏 → 触发查询
2. **粒度切换** (分时/日/月) → 重新请求数据 → 切换表格列头
3. **添加对比指标** → 图表叠加新系列 → 表格增加列
4. **导出Excel** → 调用后端导出接口 → 下载当前查询结果
5. **双击表格单元格** → 弹出该时间点的所有关联指标

## 六、图表类型映射规则

```typescript
function getChartType(categoryL1: string, categoryL2: string): string {
  if (categoryL1 === '排名') return 'horizontalBar'
  if (categoryL1 === '同比' || categoryL1 === '环比') return 'lineBarMix'
  if (categoryL1 === '负荷率' || categoryL1 === '覆盖率') return 'area'
  if (categoryL1 === '电费') return 'bar'
  if (categoryL1 === '电量' || categoryL1 === '电价') return 'line'
  if (categoryL1 === '边际贡献') return 'line'
  return 'line'
}
```

## 七、关键交互流程

### 流程1: 打开看板 → 看到数据

```
用户点击「指标中心」菜单
  → Vue Router 跳转 /indicator/dashboard
  → Pinia store 加载5个KPI指标
  → 并行请求: GET /api/indicator/data/daily?code=MLT_CONTRACT_ENERGY&latest=1
  → 并行请求: GET /api/indicator/data/compare?code=MLT_CONTRACT_ENERGY&type=yoy
  → 渲染KPI卡片
  → 异步加载图表数据(不阻塞KPI显示)
```

### 流程2: 点击树节点 → 查看指标

```
用户点击左侧「日前偏差电量」节点
```

### 流程3: 管理员新增计算指标

```
管理员进入 配置 → 指标定义 → 点击新增
  → 填写: 编码/名称/市场/分类/单位/粒度
  → 切换到「计算公式」Tab
  → 选择公式类型: 四则运算
  → 输入公式: {DAM_CLEARING_ENERGY} - {MLT_CONTRACT_ENERGY}
  → 系统自动提取依赖: DAM_CLEARING_ENERGY, MLT_CONTRACT_ENERGY
  → 自动确定计算层级: max(依赖层级)+1 = 2
  → 渲染依赖拓扑图
  → 点击「公式预览」→ 选择日期+机组 → 计算结果
  → 点击「保存」→ POST /api/indicator/definitions
  → 跳回列表页
```
