# 指标中心 - 电力营销管控系统

> 面向电力交易市场的多省网一体化指标管理平台，覆盖199个电力营销指标的统一管理、配置化计算和多维分析。

## 核心指标

- **199个指标**：覆盖电量/电价/电费/偏差/边际贡献/成本/经营效率
- **10层计算拓扑**：原始数据 → 基础计算 → 偏差 → 电费 → 差价合约 → 边际贡献 → 负荷率 → 综合指标 → 同环比 → 排名
- **9个省网**：山西/河南/湖北/蒙西/蒙东/宁夏/甘肃/陕西/云南/广西
- **配置化计算**：新增指标零代码，公式表达式驱动

## 技术栈

| 层 | 技术 |
|---|------|
| 后端 | Spring Boot 3.x + Spring Cloud + MyBatis Plus |
| 前端 | Vue 3 + Element Plus + ECharts |
| 数据库 | PostgreSQL (11个业务库 + 指标库) |
| 缓存 | Redis |
| 注册中心 | Nacos |
| 部署 | Docker Compose |
